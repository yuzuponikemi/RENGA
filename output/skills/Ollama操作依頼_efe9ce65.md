# 🔷 スキル定義: Ollama操作依頼

**説明**: Ollamaモデルの操作やトラブル診断を依頼するスキル
**ステータス**: 🔒 審査中 | **ソース数**: 4 件 | **カテゴリ**: Ollama操作依頼

---

## テンプレートプロンプト（抽象）

```
{ollamaコマンド} {オプション引数} {追加指示}
```

## 変数一覧

- `{{ollamaコマンド}}`: 実行するOllamaコマンド（例: ollama run, ollama pull）
- `{{オプション引数}}`: モデル名やフラグなどの引数（例: モデル名）
- `{{追加指示}}`: ユーザーからの具体的な依頼内容（例: プルしておいて、診断して）

## 活用シーン

- 特定のモデルをダウンロードしてほしいとき
- モードで応答が得られなかったときの診断依頼
- モデルの動作確認やログ調査を依頼するとき

# 🔶 具体インスタンス

**変数の値**
  - `ollamaコマンド`: ollama run
  - `オプション引数`: {モデル名}
  - `追加指示`: これを使うと特定のOSでは速くなるらしいので、プルしておいてもらえますか

**実際のプロンプト**
```
ollama run {モデル名} これを使うと特定のOSでは速くなるらしいので、プルしておいてもらえますか
```
