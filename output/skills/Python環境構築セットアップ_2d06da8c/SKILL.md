## テンプレートプロンプト

```
私は新しいプロジェクトを始める時に、{言語}のモダンな開発環境を毎回セットアップしています。具体的には{ツール1}での環境構築、{ツール2}、{ツール3}などの設定です。{ツール4}もかな。{コマンド}して、{設定ファイル}を整備して…その辺りをskillにまとめときたいです
```

## 変数一覧

- `{{言語}}`: 使用するプログラミング言語（例: python）
- `{{ツール1}}`: 環境構築ツール（例: uv）
- `{{ツール2}}`: リンター・フォーマッター（例: ruff）
- `{{ツール3}}`: テストフレームワーク（例: pytest）
- `{{ツール4}}`: 型チェッカー（例: mypy）
- `{{コマンド}}`: 実行するコマンド（例: uv sync）
- `{{設定ファイル}}`: 整備する設定ファイル（例: gitignore）

## 活用シーン

- 新規プロジェクトの初期環境構築を依頼する
- 既存プロジェクトにトレーシングツールを追加する
- プロジェクトのコード品質ツールを一括設定する
