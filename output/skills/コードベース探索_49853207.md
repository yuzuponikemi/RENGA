# 🔷 スキル定義: コードベース探索テンプレート

**説明**: プロジェクトの構造・アーキテクチャ・LLM利用を体系的に調査
**ステータス**: ✅ 公開中 | **ソース数**: 7 件 | **カテゴリ**: コードベース探索

---

## テンプレートプロンプト（抽象）

```
Thoroughly explore the {project_path} codebase. I need to understand:

1. The full pipeline architecture - read main entry points ({main_entry_points}), all node/module implementations
2. All modules under {source_directory} - what each does, how they interact
3. Config system - {config_files}
4. Current LLM integration patterns - how {llm_framework} is used, what models are supported
5. {specialized_pipeline} - {specialized_integration_details}
6. Data flow - what state is passed between nodes
7. Any existing extension points, plugin patterns, or hooks
8. What external services are used ({external_services})
9. The {translation_pipeline}
10. The {research_pipeline}

Read the key files thoroughly and give me a comprehensive summary of the architecture, capabilities, limitations, and natural extension points. I want to brainstorm what new features could be built on top of this system.

Focus on: {focus_files}
```

## 変数一覧

- `{{project_path}}`: 探索対象のプロジェクトの絶対パスまたは相対パス（例: /Users/yourname/projects/your-project）
- `{{main_entry_points}}`: 主要なエントリーポイントファイル名（例: main.py, __main__.py）
- `{{source_directory}}`: ソースコードが格納されているディレクトリ名（例: src/, project_name/）
- `{{config_files}}`: 設定ファイルのパスまたはパターン（例: config/*.yaml, config/settings.json）
- `{{llm_framework}}`: 使用されているLLMフレームワーク名（例: ChatOllama, LangChain）
- `{{specialized_pipeline}}`: 調査したい特殊パイプラインの名前（例: Audio pipeline, Book guide pipeline）
- `{{specialized_integration_details}}`: 特殊パイプラインの統合詳細（例: 特定の外部サービス統合の詳細）
- `{{external_services}}`: 使用されている外部サービスのカンマ区切りリスト（例: Tavily, DuckDuckGo, Ollama）
- `{{translation_pipeline}}`: 翻訳パイプラインの説明（例: The translation pipeline）
- `{{research_pipeline}}`: リサーチ/批評/拡張パイプラインの説明（例: The research/critique/enrichment pipeline）
- `{{focus_files}}`: 特に重点的に読むべきファイルのカンマ区切りリスト（例: utils/logger.py, services/*.py）

## 活用シーン

- 新しいプロジェクトに参加した際の初期コードベース理解
- 既存プロジェクトへの機能追加やリファクタリング前の現状把握
- LLM統合パターンや外部サービス利用状況の棚卸し
