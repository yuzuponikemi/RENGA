## テンプレートプロンプト

```
以下の{プロジェクト名}のコードベースを探索して、{探索目的}を理解したい。

探索対象:
- {主要ファイル・ディレクトリ}

特に以下の点に注目:
1. {注目ポイント1}
2. {注目ポイント2}
3. {注目ポイント3}
4. {注目ポイント4}

{追加指示}
```

## 変数一覧

- `{{プロジェクト名}}`: 探索するプロジェクトの名前（例: あなたのプロジェクト名）
- `{{探索目的}}`: 探索の目的（例: アーキテクチャ理解、拡張アイデアの検討、リファクタリング計画）
- `{{主要ファイル・ディレクトリ}}`: 探索すべき主要なファイルやディレクトリのリスト（例: README, src/, main.py, pipeline.py）
- `{{注目ポイント1}}`: 最初に注目すべきポイント
- `{{注目ポイント2}}`: 2番目に注目すべきポイント
- `{{注目ポイント3}}`: 3番目に注目すべきポイント
- `{{注目ポイント4}}`: 4番目に注目すべきポイント
- `{{追加指示}}`: 追加の指示や要望（例: 包括的なサマリーを提供、問題点を指摘）

## 活用シーン

- 新しいプロジェクトに参加した際の初期コードベース理解
- 既存プロジェクトのリファクタリング前の現状把握
- 特定機能の実装パターン調査
