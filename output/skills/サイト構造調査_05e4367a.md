# 🔷 スキル定義: サイト構造調査

**説明**: 指定サイトの記事構造・リンク・RSSを分析する
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **カテゴリ**: サイト構造調査

---

## テンプレートプロンプト（抽象）

```
{サイトURL} を詳しく調べてください。

1. RSSフィード {RSSフィードURL} をフェッチして、最新{記事件数}件の記事タイトル・URL・日付を列挙
2. 最新記事を1本フルでフェッチして以下を分析:
   - 本文の構造（見出し、段落数）
   - 外部リンクを全部列挙（アンカーテキスト + URL）
   - リンクをカテゴリ分類: 「{重要カテゴリ名}」vs「{ノイズカテゴリ名}」
   - 重要リンクは何本あるか

3. もう1記事（{日数範囲}日前のもの）も同様にリンク分析して、重複しているドメインや記事があるか確認

結果を具体的なURL込みで報告してください。{文字数制限}以内。
```

## 変数一覧

- `{{サイトURL}}`: 調査対象のサイトのトップページURL
- `{{RSSフィードURL}}`: RSSフィードのURL
- `{{記事件数}}`: RSSから取得する最新記事の件数
- `{{重要カテゴリ名}}`: 重要リンクの分類名
- `{{ノイズカテゴリ名}}`: ノイズリンクの分類名
- `{{日数範囲}}`: 比較対象の記事の日数範囲
- `{{文字数制限}}`: 報告の最大文字数

## 活用シーン

- ブログサイトの記事構造と外部リンク傾向を調査したいとき
- 競合サイトの情報ソースやリンク戦略を分析したいとき
- 自社サイトのRSSフィード設計やリンク分類の参考にしたいとき
