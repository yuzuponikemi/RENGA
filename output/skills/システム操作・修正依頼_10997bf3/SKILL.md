# システム操作・修正依頼

**説明**: システムの起動、再起動、修正、設定変更などを依頼するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 5 件 | **ユニークユーザー**: 5 人 | **カテゴリ**: システム操作依頼

---

## テンプレートプロンプト

```
{システム名}は起動できますか？{前提条件}は起動しておきました。{追加操作}できますか？{詳細情報}。{問題内容}を修正できますか？
```

## 変数一覧

- `{{システム名}}`: 操作対象のシステムやツール名（例: データベース, Docker, メールサーバー）
- `{{前提条件}}`: すでに実施済みの操作や状態（例: ベースシステムは起動しておきました）
- `{{追加操作}}`: 依頼する追加の操作（例: 再起動してフルインデックス）
- `{{詳細情報}}`: 具体的な設定や識別情報（例: メールアドレス、サブドメイン名）
- `{{問題内容}}`: 発生している問題や修正してほしい内容（例: ビルド失敗、動かない）

## 活用シーン

- データベースの起動や再起動を依頼する
- メール設定やサブドメイン変更を依頼する
- ビルド失敗や動作不良の修正を依頼する
