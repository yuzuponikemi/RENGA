# 🔷 スキル定義: タスク再開・継続依頼

**説明**: 中断した作業を再開し、追加情報を反映して継続実行するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **カテゴリ**: タスク再開・継続依頼

---

## テンプレートプロンプト（抽象）

```
前回やっていた {タスク名} を再開してほしい
{進捗状況}

今回の追加情報は以下の通りです：
{追加情報}

これを使って、以下の手法で分析・実行してください：
{実行手順}

まず {事前準備} をしてから、実際にやってみてください。
```

## 変数一覧

- `{{タスク名}}`: 再開したい作業の名前（例: 動画の強制アライメント処理）
- `{{進捗状況}}`: 前回の進捗（チェックリスト形式など）
- `{{追加情報}}`: 今回新たに提供する情報（例: 歌詞テキスト、設定値など）
- `{{実行手順}}`: 具体的なコマンドや手法
- `{{事前準備}}`: 実行前に行うべき準備

## 活用シーン

- 動画の強制アライメント処理を歌詞付きで再開する
- ソースコード解析タスクを中断箇所から再開する
- 機械学習モデルの学習を追加データで再開する
