# 🔷 スキル定義: タスク進捗確認と指示

**説明**: AIツールの状態確認・再起動・リポジトリ更新を指示するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 5 件 | **カテゴリ**: タスク進捗確認と指示

---

## テンプレートプロンプト（抽象）

```
{ツール名}の状態を確認し、必要に応じて{操作内容}を実行してください。{追加指示}。また、{リポジトリ名}の{変更内容}を行いましたので、{後続処理}をお願いします。
```

## 変数一覧

- `{{ツール名}}`: 確認・操作対象のツール名（例: 特定のAIツール、ミドルウェア）
- `{{操作内容}}`: 実行する操作（例: 再起動, プロセス確認, リソース削除）
- `{{追加指示}}`: 補足的な指示（例: 処理中のプロセスに注意, 優先度指定）
- `{{リポジトリ名}}`: 更新対象のGitリポジトリ名（例: 特定のプロジェクトリポジトリ）
- `{{変更内容}}`: リポジトリに行った変更（例: 改良, 機能追加, 公開設定）
- `{{後続処理}}`: 変更後の後続アクション（例: 再起動, テスト実行, 公開）

## 活用シーン

- 特定のツールのバージョンアップ後の再起動と動作確認を依頼する
- リソース解放のために不要なデータ削除を指示する
- リモートリポジトリの更新を反映するための再起動を依頼する
