# 🔷 スキル定義: テスト実行・修正依頼

**説明**: 特定のテストコマンドを実行し、結果に応じて修正を依頼する
**ステータス**: 🔒 審査中 | **ソース数**: 3 件 | **カテゴリ**: テスト実行・修正依頼

---

## テンプレートプロンプト（抽象）

```
{command} を実行して動作確認してください。その後、{issue_description} について修正してください。
```

## 変数一覧

- `{{command}}`: 実行するテストコマンド（例: 特定のテストスクリプトを指定）
- `{{issue_description}}`: 修正が必要な課題や変更内容の説明（例: 特定のテストで共通の設定が使えるようにする）

## 活用シーン

- テストスクリプトの動作確認と修正依頼
- 特定のテストケースを実行し、エラーがあれば修正を依頼
- テスト環境のセットアップ確認と設定変更依頼
