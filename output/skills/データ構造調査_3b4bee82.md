# 🔷 スキル定義: データ構造調査スキル

**説明**: 特定のデータソースの構造・形式・取得方法を調査する汎用テンプレート
**ステータス**: ✅ 公開中 | **ソース数**: 7 件 | **カテゴリ**: データ構造調査

---

## テンプレートプロンプト（抽象）

```
Thoroughly research the structure of {target_data_source}. I need to understand:

1. What data is included in a typical {data_source_type} export (especially: {key_data_categories})
2. The file formats used ({expected_formats})
3. The directory structure of a {export_name} export
4. Specifically for data that reveals "{analysis_goal}" - which {data_source_type} sources are most valuable:
   - {source_1} - reveals {source_1_value}
   - {source_2} - reveals {source_2_value}
   - {source_3} - reveals {source_3_value}
   - {source_4} - reveals {source_4_value}
   - {source_5} - reveals {source_5_value}
   - {source_6} - reveals {source_6_value}

Focus on the {primary_format} structure of {primary_sources} as these are the richest sources for {analysis_goal}.

Search the web for "{search_query}" and related queries to get current information.
```

## 変数一覧

- `{{target_data_source}}`: 調査対象のデータソース名（例: 特定のサービスからのデータエクスポート、API、ログファイル）
- `{{data_source_type}}`: データソースの種類（例: データエクスポート、API、ログファイル）
- `{{key_data_categories}}`: 特に注目するデータカテゴリのリスト（例: 検索履歴、チャット履歴、位置情報）
- `{{expected_formats}}`: 想定されるファイル形式（例: JSON、HTML、MBOX、CSV）
- `{{export_name}}`: エクスポートの名称（例: 特定のサービスエクスポート、セッションログ）
- `{{analysis_goal}}`: 分析の目的（例: 思考・興味・感情の抽出、会話スレッドの追跡）
- `{{source_1}}`: データソース1の名前
- `{{source_1_value}}`: データソース1が明らかにする情報
- `{{source_2}}`: データソース2の名前
- `{{source_2_value}}`: データソース2が明らかにする情報
- `{{source_3}}`: データソース3の名前
- `{{source_3_value}}`: データソース3が明らかにする情報
- `{{source_4}}`: データソース4の名前
- `{{source_4_value}}`: データソース4が明らかにする情報
- `{{source_5}}`: データソース5の名前
- `{{source_5_value}}`: データソース5が明らかにする情報
- `{{source_6}}`: データソース6の名前
- `{{source_6_value}}`: データソース6が明らかにする情報
- `{{primary_format}}`: 主に分析するファイル形式（例: JSON、JSONL）
- `{{primary_sources}}`: 主な分析対象ソース（例: アクティビティログ、セッションログ）
- `{{search_query}}`: Web検索に使用するクエリ

## 活用シーン

- 特定のサービスからユーザーの思考パターンを抽出するためのデータ構造調査
- チャット履歴のスレッド情報を取得する方法の調査
- 開発ツールのセッションログ保存動作とログ管理ポリシーの調査
