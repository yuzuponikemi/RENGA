# 🔷 スキル定義: ノートブック改善・拡張指示

**説明**: 既存ノートブックの内容改善や拡張を指示するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 11 件 | **カテゴリ**: ノートブック改善指示

---

## テンプレートプロンプト（抽象）

```
{ノートブック識別子}について、{具体的な改善内容}を行ってください。特に{注意点や補足要件}に気をつけて、{参照すべき既存ノートブックや基準}を参考にしながら実装してください。
```

## 変数一覧

- `{{ノートブック識別子}}`: 対象となるノートブックの番号や名前（例: ノートブック09, unit0.3のノートブック）
- `{{具体的な改善内容}}`: 行ってほしい変更の詳細（例: 教科書的な解説を充実、エラー修正、特定手法の拡張実装）
- `{{注意点や補足要件}}`: 実装時に気をつけるべきこと（例: セルの種類を間違えない、番号が被らないようにする）
- `{{参照すべき既存ノートブックや基準}}`: 参考にする既存のノートブックやカリキュラム（例: ノートブック06のようなスタイル、カリキュラムに準拠）

## 活用シーン

- 既存ノートブックの解説不足を補い、教科書的に充実させる
- 特定の分析手法を深掘りしてノートブックを拡張する
- エラーが発生しているノートブックの調査と修正を依頼する

**変数の値**
  - `ノートブック識別子`: ノートブック09

  - `注意点や補足要件`: セルの種類をマークダウンとコードで間違えないようにする

**実際のプロンプト**
```
{ノートブック識別子}について、コードが多くて説明が少ないので、何をしているのか背景など踏まえて教科書的な解説をもっと充実させてください。特にセルの種類をマークダウンとコードで間違えないように気をつけて、{参照すべき既存ノートブックや基準}のようなスタイルを参考にしながら実装してください。
```
