# ノートブック解説強化

**説明**: 既存ノートブックの説明を充実させ、教科書的に改善するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 11 件 | **ユニークユーザー**: 8 人 | **カテゴリ**: ノートブック解説強化

---

## テンプレートプロンプト

```
ノートブック{ノートブック番号}の内容が分かりづらいので、コードの説明や背景を踏まえて、{参照ノートブック番号}のように教科書的な解説を充実させてください。また、{注意事項}に気をつけて編集してください。
```

## 変数一覧

- `{{ノートブック番号}}`: 改善対象のノートブック番号（例: 09）
- `{{参照ノートブック番号}}`: 解説スタイルの参考にするノートブック番号（例: 06）
- `{{注意事項}}`: 編集時に気をつけるべき点（例: セルタイプの誤設定に注意、番号の重複を避ける）

## 活用シーン

- コード量が多く説明が少ないノートブックに詳細な解説を追加したいとき
- 特定のノートブックの解説スタイルを他のノートブックにも適用したいとき
- 新しく作成するノートブックに教科書的な解説を最初から入れたいとき
