# 🔷 スキル定義: バックグラウンドタスク監視

**説明**: 非同期処理の進捗を確認し、状態に応じてアクションを実行する
**ステータス**: ✅ 公開中 | **ソース数**: 12 件 | **カテゴリ**: バックグラウンドタスク監視

---

## テンプレートプロンプト（抽象）

```
Check the status of background task {task_id} ({task_name}). Use TaskOutput with block=false to see current progress. If {first_step} is still running, schedule another wakeup. If {first_step} finished and {second_step} started, report progress. If the full pipeline finished, read the output, summarize results, and push a notification to the user.
```

## 変数一覧

- `{{task_id}}`: 監視対象のタスクID
- `{{task_name}}`: タスクの説明（例: パイプライン名と処理内容）
- `{{first_step}}`: 最初の処理ステップ名
- `{{second_step}}`: 次の処理ステップ名

## 活用シーン

- 音声・動画処理パイプラインの進捗監視
- データ変換・エンコード処理の状態確認と通知
- バッチジョブの完了監視と結果通知
