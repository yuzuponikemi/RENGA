# バッチ結果確認

**説明**: バックグラウンドタスクの出力結果を確認する
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **ユニークユーザー**: 1 人 | **カテゴリ**: バッチ結果確認

---

## テンプレートプロンプト

```
{作業内容の説明}の結果を確認してください。バックグラウンドタスクID: {タスクID}、出力ファイル: {出力ファイルパス}
```

## 変数一覧

- `{{作業内容の説明}}`: 実行したバッチ処理の内容や目的（例: バッチテスト）
- `{{タスクID}}`: バックグラウンドタスクを識別するID（例: タスクID文字列）
- `{{出力ファイルパス}}`: 結果が保存されているファイルのフルパス（例: /private/tmp/.../tasks/{タスクID}.output）

## 活用シーン

- バッチ処理の実行結果を確認したいとき
- 長時間処理の完了状態をチェックするとき
- 複数のバッチ処理結果をまとめて確認するとき
