# 🔷 スキル定義: パイプライン分析・構築

**説明**: データパイプラインの課題分析、設計、実行指示を行うスキル
**ステータス**: ✅ 公開中 | **ソース数**: 8 件 | **カテゴリ**: パイプライン分析・構築

---

## テンプレートプロンプト（抽象）

```
{パイプラインの種類}について、{分析対象}を{アクション}してください。{追加条件}があれば考慮し、{リソース制約}の範囲で{実行範囲}を処理してください。
```

## 変数一覧

- `{{パイプラインの種類}}`: 分析・構築するパイプラインの名称や種類（例: 動画文字起こしパイプライン、記事作成パイプライン）
- `{{分析対象}}`: パイプライン内で処理する具体的なデータやリソース（例: 特定の動画URL、既存のパイプライン構成）
- `{{アクション}}`: 実行してほしい具体的な操作（例: 課題を見つける、現状をまとめる、設計する、実行する）
- `{{追加条件}}`: 処理に含めるべき追加のステップや要件（例: ソースチェック、ファクトチェック、考察追加）
- `{{リソース制約}}`: 処理実行時の制約条件（例: リソースが空いたら実行、止めていい処理）
- `{{実行範囲}}`: 処理を適用する範囲や対象（例: 最終回だけ、各記事にフルパイプライン）

## 活用シーン

- 既存のデータパイプラインの課題を発見し改善提案をする
- 新しいコンテンツ生成パイプラインをゼロから設計する
- 複数のパイプラインを統合して効率化する

# 🔶 具体インスタンス

### インスタンス 1: 動画パイプライン課題分析

> 元プロンプト: *指定した動画URLをプロセスしてみて、このパイプラインの課題を見つけてみて*

**変数の値**
  - `パイプラインの種類`: 動画文字起こしパイプライン
  - `分析対象`: 指定した動画URL
  - `アクション`: 課題を見つける
  - `追加条件`: なし
  - `リソース制約`: なし
  - `実行範囲`: その動画のみ

**実際のプロンプト**
```
動画文字起こしパイプラインについて、指定した動画URLを課題を見つけてください。特に制約はなく、その動画のみを処理してください。
```

---

### インスタンス 2: フルパイプライン設計と実行

> 元プロンプト: *おーけーありがとう　では文字起こし→話されている内容の要約　ソースチェック　記事化　みたいな流れでパイプライン設計して　最終回だけ処理してみましょう　リソースが空いたら実行で　まだ始まってない別の処理は止めていいです*

**変数の値**
  - `パイプラインの種類`: 記事作成パイプライン
  - `分析対象`: 文字起こし→話されている内容の要約→ソースチェック→記事化
  - `アクション`: 設計する
  - `追加条件`: 最終回だけ処理
  - `リソース制約`: リソースが空いたら実行、まだ始まってない別の処理は止める
  - `実行範囲`: 最終回のみ

**実際のプロンプト**
```
記事作成パイプラインについて、文字起こし→話されている内容の要約→ソースチェック→記事化の流れで設計してください。最終回だけ処理してください。リソースが空いたら実行で、まだ始まってない別の処理は止めていいです。
```

---

### インスタンス 3: パイプライン統合指示

> 元プロンプト: *もしかしていまつて、トリプル抽出だけじゃなくて記事作成とファクトチェック含むフルパイプラインを各記事に実行ちゅう？　なにをしてるの*

**変数の値**
  - `パイプラインの種類`: トリプル抽出・記事作成・ファクトチェックを含むフルパイプライン
  - `分析対象`: 各記事
  - `アクション`: 統合する
  - `追加条件`: 現在の実行状況を確認
  - `リソース制約`: なし
  - `実行範囲`: 各記事

**実際のプロンプト**
```
トリプル抽出・記事作成・ファクトチェックを含むフルパイプラインについて、各記事に統合してください。現在の実行状況を確認し、何をしているか教えてください。
```
