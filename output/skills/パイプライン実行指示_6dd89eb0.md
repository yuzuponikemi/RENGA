# 🔷 スキル定義: パイプライン実行指示

**説明**: 指定された資料に対して分析パイプラインを実行し、成果物を生成する
**ステータス**: ✅ 公開中 | **ソース数**: 9 件 | **カテゴリ**: パイプライン実行指示

---

## テンプレートプロンプト（抽象）

```
{資料名}について、{パイプライン名}のパイプラインで{実行内容}まで実行してください。{追加指示}
```

## 変数一覧

- `{{資料名}}`: 分析対象の書籍・文書・ファイル名（例: なめらかな社会とその敵）
- `{{パイプライン名}}`: 実行するパイプラインの種類（例: web research, test-smith）
- `{{実行内容}}`: パイプラインの最終出力段階（例: ポッドキャスト生成, ブックガイド作成, 動作確認）
- `{{追加指示}}`: その他の補足条件や要望（例: 日本語で保存, ステップごとに入力と出力を確認）

## 活用シーン

- 新規資料の分析パイプラインを一括実行したいとき
- 特定のパイプラインの動作確認やテストを依頼するとき
- 中間成果物の言語や保存形式を指定して再実行したいとき
