# 🔷 スキル定義: ノートブックプロット日本語英訳

**説明**: Jupyterノートブックのプロット要素の日本語を英語に翻訳するスキル
**ステータス**: 🔒 審査中 | **ソース数**: 10 件 | **カテゴリ**: プロット要素の日英翻訳

---

## テンプレートプロンプト（抽象）

```
You need to translate Japanese text in matplotlib/seaborn plot elements to English in Jupyter notebooks.

IMPORTANT RULES:
- Only modify CODE cells, not markdown cells
- Only modify strings that appear in plot-related function calls ({plot_function_list})
- Do NOT modify comments (lines starting with #) or print() statements
- Do NOT modify docstrings
- Keep the code structure exactly the same, only change the Japanese string content to English
- Do NOT use japanize_matplotlib

Process these notebooks in {base_directory}:
{notebook_list}

For each notebook:
1. Read it to find all Japanese text in plot-related code lines
2. Write a Python script that does exact string replacement for each Japanese string found
3. Run the script

The approach: write a single Python script with a translation map covering all Japanese strings in these notebooks, then run it. The script should:
- Load each notebook as JSON
- Iterate through code cells
- For each line, check if it's plot-related (not comment, not print)
- Replace Japanese strings with English equivalents
- Save the notebook back

Make sure translations are accurate and natural English. Be thorough - find EVERY Japanese string in plot-related lines.

After running the script, verify by counting remaining Japanese in plot lines for each notebook. Re-run if needed to get the count to 0.

{additional_instructions}
```

## 変数一覧

- `{{plot_function_list}}`: 翻訳対象とするプロット関連関数のリスト（例: set_title, set_xlabel, set_ylabel, suptitle, .text(), label=, annotate, Patch(, axhline, legend, fig.text, set_xticklabels, plt.title, plt.xlabel, plt.ylabel）
- `{{base_directory}}`: ノートブックファイルが格納されているベースディレクトリのパス
- `{{notebook_list}}`: 処理対象のノートブックファイル一覧（ファイル名とインスタンス数のリスト）
- `{{additional_instructions}}`: 追加の指示や注意事項（例: 特定の翻訳パターン、共通の用語集、スキャン結果ファイルの参照など）

## 活用シーン

- チーム内で共有するノートブックの国際化対応
