# 🔷 スキル定義: モデル設定調整

**説明**: タイムアウトやモデル変更など、モデル設定を調整するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 5 件 | **カテゴリ**: モデル設定調整

---

## テンプレートプロンプト（抽象）

```
うーん　まだタイムアウトしてるっぽいですね　{モデル名} {モデルID} {メモリサイズ} にしてみましょうか・・・　あとタイムアウトは{タイムアウト秒数}秒じゃ足りないらしい・・・？　ログを見て確認してみて
```

## 変数一覧

- `{{モデル名}}`: 使用するAIモデルの名前（例: llama3.2:latest）
- `{{モデルID}}`: モデルのコンテナID（例: a80c4f17acd5）
- `{{メモリサイズ}}`: モデルに割り当てるメモリ容量（例: 2.0 GB）
- `{{タイムアウト秒数}}`: タイムアウトの秒数（例: 1800）

## 活用シーン

- タイムアウトエラーが発生した際のモデル設定変更
- メモリ不足による動作不良時のモデル切り替え
- 別のモデルに変更してパフォーマンス改善を試す場合
