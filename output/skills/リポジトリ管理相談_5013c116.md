## テンプレートプロンプト（抽象）

```
この{リポジトリ名}リポジトリだけど、git的にはどう管理すればいいかな　すでに{フォーク状態}してるから普通にPushすると自分のものとして開発できる？　{注意事項}ので迷惑かけたくない
```

## 変数一覧

- `{{リポジトリ名}}`: 管理対象のリポジトリ名（例: あなたのリポジトリ名）
- `{{フォーク状態}}`: フォーク済みかどうかの状態（例: フォーク済み）
- `{{注意事項}}`: 気にしている点（例: OSSプロジェクト）

## 活用シーン

- フォークしたリポジトリを自分の管理下で開発したい時
- 新規フォルダをGitリポジトリ化して管理したい時
- アクセス権限を特定パスに限定したい時
