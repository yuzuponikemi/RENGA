# 🔷 スキル定義: ワークフロー設計相談

**説明**: Claude Code等のAIツールを使った作業環境やワークフロー設計を相談・検証する
**ステータス**: ✅ 公開中 | **ソース数**: 9 件 | **カテゴリ**: ワークフロー設計相談

---

## テンプレートプロンプト（抽象）

```
私は{ツール名}を使って{目的}を実現したいと考えています。現在の環境は{現在の環境}で、{課題}があります。{具体的な質問}について教えてください。なお、{追加条件}を考慮してください。
```

## 変数一覧

- `{{ツール名}}`: 使用しているAIツールやアプリケーション名（例: Claude Code, Gemini, VS Code）
- `{{目的}}`: 実現したいことや達成したいゴール（例: 情報整理プロジェクトのワークスペース構築）
- `{{現在の環境}}`: 現在のセットアップやフォルダ構成（例: 全リポジトリが一つのフォルダに置いてある）
- `{{課題}}`: 直面している問題や疑問点（例: ワークスペース機能がよくわからない）
- `{{具体的な質問}}`: 具体的に聞きたいこと（例: このやり方であっているか？）
- `{{追加条件}}`: 考慮すべき追加情報や制約（例: ネットワークがない場所を移動する場合）

## 活用シーン

- 新しいプロジェクトの作業環境をAIツールと連携して設計したいとき
- 既存のワークフローにAIエージェントを組み込む方法を相談したいとき
- AIツールのログ管理やセッション管理の仕様を確認したいとき

  - `追加条件`: 今いらないプロジェクトも残っている
