# 🔷 スキル定義: 会話ログからの記事生成

**説明**: 会話ログや調査内容から時系列記事を生成し公開する
**ステータス**: ✅ 公開中 | **ソース数**: 6 件 | **カテゴリ**: 会話からの記事生成

---

## テンプレートプロンプト（抽象）

```
以下の会話ログや調査内容を参照しながら、{記事テーマ}に関する時系列記事を生成してください。記事は{公開先}に掲載する想定です。{記事の長さ}程度の文字数で、{文体や構成の条件}を守ってください。論点は厳選し、同じ構成の繰り返しを避け、自然な文体で一つの論点を深掘りして詳細に論じてください。
```

## 変数一覧

- `{{記事テーマ}}`: 記事の主題や内容（例: 開発記、調査結果）
- `{{公開先}}`: 記事を掲載するサービス名（例: 社内ブログ、公開プラットフォーム）
- `{{記事の長さ}}`: 目標文字数（例: 8000文字、適量）
- `{{文体や構成の条件}}`: 文体や構成に関する追加指示（例: 自然な文体、論点を厳選、繰り返しを避ける）

## 活用シーン

- 開発プロジェクトの会話ログから時系列の開発記を生成する
- 日々の調査内容をまとめた記事を社内ブログに掲載する
- RSSで収集した情報を深掘りし、重要なリンクをまとめたレポートを作成する
