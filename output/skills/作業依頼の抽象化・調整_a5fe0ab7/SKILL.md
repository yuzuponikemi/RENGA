# 作業依頼の抽象化・調整

**説明**: ユーザーの曖昧な作業依頼を具体化し、汎用テンプレートに変換するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 4 件 | **ユニークユーザー**: 4 人 | **カテゴリ**: 情報抽出・要約依頼

---

## テンプレートプロンプト

```
{依頼内容}について、{対象物}を{具体的なアクション}してほしいです。{追加条件}があれば教えてください。また、{補足情報}も考慮して{成果物の形式}で出力してください。
```

## 変数一覧

- `{{依頼内容}}`: ユーザーが依頼する作業の大まかなテーマ（例: ブログポストのピックアップ）
- `{{対象物}}`: 作業の対象となる具体的なもの（例: 公開ブログポスト）
- `{{具体的なアクション}}`: 対象物に対して行う操作（例: トピックに近いものをピックアップ）
- `{{追加条件}}`: 作業に付随する条件や要望（例: 必要な操作があれば教える）
- `{{補足情報}}`: 作業を進める上で考慮すべき追加情報（例: ゲストのプロフィール）
- `{{成果物の形式}}`: 最終的な出力の形式やスタイル（例: 文字数指定、Web検索結果の反映）

## 活用シーン

- 公開ブログ記事やポッドキャストから特定トピックの情報を抽出・要約する
- テスト実行やコード修正の依頼を具体化する
- データセットやシステムの調整依頼を明確にする

## 具体インスタンス

### インスタンス 1: ブログポスト抽出

> 元プロンプト: *おけです　かれの公開ブログポストのなかで、今回のトピックにちかいものをピックアップしてほしいです*

**変数の値**
  - `依頼内容`: 公開ブログポストのピックアップ
  - `対象物`: 公開ブログポスト
  - `具体的なアクション`: 指定したトピックに近いものをピックアップ
  - `追加条件`: 必要な操作があれば教える
  - `補足情報`: 特に追加条件なし
  - `成果物の形式`: リスト形式

**実際のプロンプト**
```
あなたの公開ブログポストについて、指定したトピックに近いものをピックアップしてほしいです。必要な操作があれば教えてください。また、特に追加条件はありませんが、出力はリスト形式でお願いします。
```

---

### インスタンス 2: ポッドキャストプロフィール追加

> 元プロンプト: *あと、この２つのポッドキャストに関しては　ゲストのプロフィールも最初に添えてあると話が入ってきやすいです　何文字くらいがいいかな　あと　ゲストのプロフィールを知るにはWeb検索も必要だから難しいよね　まずダリオであなたが作成してみて*

**変数の値**
  - `依頼内容`: ポッドキャストの要約とプロフィール追加
  - `対象物`: 指定した2つのポッドキャスト
  - `具体的なアクション`: ゲストのプロフィールを最初に添えて要約
  - `追加条件`: 何文字くらいがいいか相談、Web検索が必要な点を考慮
  - `補足情報`: まずあなたが作成してみる
  - `成果物の形式`: プロフィール付きの要約形式

**実際のプロンプト**
```
指定した2つのポッドキャストについて、ゲストのプロフィールも最初に添えてあると話が入ってきやすいです。何文字くらいがいいかな？また、ゲストのプロフィールを知るにはWeb検索も必要だから難しいよね。まずあなたが作成してみて。出力はプロフィール付きの要約形式でお願いします。
```

### インスタンス 3: データセット修正依頼

> 元プロンプト: *うーむ　元々小さいデータセットでうまくいくことはわかっていたので、本番用でうあくいくように修正して欲しいです*

**変数の値**
  - `依頼内容`: データセットの修正
  - `対象物`: 元々小さいデータセット
  - `具体的なアクション`: 本番用でうまくいくように修正
  - `追加条件`: 必要な操作があれば教える
  - `補足情報`: 本番環境の仕様を考慮
  - `成果物の形式`: 修正後のコード形式

**実際のプロンプト**
```
元々小さいデータセットでうまくいくことはわかっていたので、本番用でうまくいくように修正して欲しいです。必要な操作があれば教えてください。また、本番環境の仕様を考慮して、出力は修正後のコード形式でお願いします。
```
