# 🔷 スキル定義: 作業環境設定の改善依頼

**説明**: AIエージェントの動作設定やルールを改善するためのテンプレート
**ステータス**: 🔒 審査中 | **ソース数**: 4 件 | **カテゴリ**: 作業環境の設定改善

---

## テンプレートプロンプト（抽象）

```
今後はセッション開始時に {設定ファイル名} が参照先を示していたら、必ずそのファイルも読んでから作業を始めます。これを確証できるように、{設定ファイル名} に厳しく書いておいてください。また以下の部分の作業が最も重要だったのですが、まだスキップしてますね。# {対象設定項目} について

{設定内容の説明}
```

## 変数一覧

- `{{設定ファイル名}}`: AIエージェントが参照すべき設定ファイルの名前（例: プロジェクトルールファイル）
- `{{対象設定項目}}`: 改善・確認が必要な設定項目の名前（例: プロジェクト設定ファイル）
- `{{設定内容の説明}}`: その設定項目に関する具体的な説明やルール（例: サイト名、テーマ、ナビゲーション、プラグイン、拡張機能の中央設定ファイルです。一部のコンテンツは自動検出されますが、その他のコンテンツは手動で追加が必要です。）

## 活用シーン

- AIエージェントが設定ファイルを読み飛ばす癖がある場合の改善依頼
- 特定の設定項目の作業がスキップされている場合の再実行依頼
- 出力形式や表示内容の改善を依頼する場合
