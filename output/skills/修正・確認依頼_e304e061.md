# 🔷 スキル定義: 修正確認ループ依頼

**説明**: 修正→確認→再修正のサイクルを繰り返すスキル
**ステータス**: ✅ 公開中 | **ソース数**: 7 件 | **カテゴリ**: 修正・確認依頼

---

## テンプレートプロンプト（抽象）

```
{対象}に対して{修正内容}を適用し、{確認項目}を確認してください。問題があれば{再試行条件}になるまで修正と確認を繰り返してください。{追加指示}
```

## 変数一覧

- `{{対象}}`: 修正・確認の対象（例: 記事、システム、設定）
- `{{修正内容}}`: 具体的な修正指示（例: 色を直す、設定を変更する、内容を改善する）
- `{{確認項目}}`: 確認すべきポイント（例: 色が正しいか、返信が来たか、レポートが改善されたか）
- `{{再試行条件}}`: 修正と確認を繰り返す終了条件（例: 成功するまで、問題がなくなるまで）
- `{{追加指示}}`: その他の補足指示（例: 特定の言葉を含める、ログを見ながら実行）

## 活用シーン

- Webサイトのデザイン修正と確認を繰り返す
- メール自動返信システムの動作確認と修正
- AIモデルの出力品質チェックと改善ループ
