# 再起動指示

**説明**: システムやアプリケーションの再起動を依頼するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **ユニークユーザー**: 1 人 | **カテゴリ**: 再起動指示

---

## テンプレートプロンプト

```
{対象}を再起動して{目的}を反映させてください
```

## 変数一覧

- `{{対象}}`: 再起動するシステムやアプリケーション名（例: サーバー、アプリ、PC）
- `{{目的}}`: 再起動によって達成したいこと（例: 変更、設定更新、アップデート）

## 活用シーン

- システム設定変更後に再起動を依頼する
- ソフトウェアアップデート後に再起動を促す
- トラブルシューティングの一環として再起動を指示する
