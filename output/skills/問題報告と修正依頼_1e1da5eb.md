# 🔷 スキル定義: 問題報告と修正依頼

**説明**: 不具合や意図しない動作を報告し、具体的な修正を依頼するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 5 件 | **カテゴリ**: 問題報告と修正依頼

---

## テンプレートプロンプト（抽象）

```
{製品名}において、{問題の内容}という問題が発生しています。期待する動作は{期待動作}です。{追加情報}。修正可能であれば対応をお願いします。
```

## 変数一覧

- `{{製品名}}`: 問題が発生しているサービスやアプリケーション名（例: ポッドキャスト、メールシステム、ブログ）
- `{{問題の内容}}`: 具体的に何が起きているか、または何が間違っているか（例: 名前が間違っている、文字数が足りない、ビルドが失敗している）
- `{{期待動作}}`: 本来あるべき正しい状態や動作（例: 人物チェックが機能する、14000字で終わる、公開ボタンがある）
- `{{追加情報}}`: 修正に必要な補足情報や希望する修正内容（例: メールアドレス、サブドメイン名、新しい名前）

## 活用シーン

- Webアプリの表示名や設定が意図と異なる場合の修正依頼
- 自動処理（ビルド、文字数制限など）が期待通り動作しない場合の報告
- 機能の不足（公開ボタンがない、チェック機能が動かないなど）を指摘して改善を依頼
