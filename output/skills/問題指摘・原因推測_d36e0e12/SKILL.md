# 問題指摘・原因推測

**説明**: ユーザーの問題を指摘し、考えられる原因を推測するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 4 件 | **ユニークユーザー**: 4 人 | **カテゴリ**: 問題指摘・原因推測

---

## テンプレートプロンプト

```
{期待結果}を目指していたのに、{実際の結果}になってしまった。これは{問題箇所}が原因かもしれない。{改善提案}は可能ですか？
```

## 変数一覧

- `{{期待結果}}`: 本来あるべき状態や目標値（例: 目標文字数、正しい名前）
- `{{実際の結果}}`: 現状の状態や数値（例: 実際の文字数、誤った名前）
- `{{問題箇所}}`: 原因と思われる機能や設定（例: 文字数制限、人物チェック機能）
- `{{改善提案}}`: ユーザーが希望する修正や確認方法（例: 名前を直す、出力文字数を調整する）

## 活用シーン

- ブログ記事の文字数が目標に届かない原因を推測する
- 名前や表記の誤りがなぜ発生したか原因を探る
- アプリの機能が期待通り動作しない理由を考える

## 具体インスタンス

### インスタンス 1: 文字数不足の原因推測

> 元プロンプト: *うーん　{目標文字数}を目指していたのに、{実際の文字数}で終わってしむった　これはなんでかな？*

**変数の値**
  - `期待結果`: {目標文字数}
  - `実際の結果`: {実際の文字数}
  - `問題箇所`: 文字数制限や生成設定
  - `改善提案`: 出力文字数を調整する

**実際のプロンプト**
```
{目標文字数}を目指していたのに、{実際の文字数}で終わってしまった。これは文字数制限や生成設定が原因かもしれない。出力文字数を調整することは可能ですか？
```

---

### インスタンス 2: 名前誤りの原因推測

> 元プロンプト: *おーけー　名前などが間違っているのですが、ほかのポッドキャストでできていた人物チェックとかが機能していないのかな？*

**変数の値**
  - `期待結果`: 正しい名前で表示される
  - `実際の結果`: {誤った名前}という名前が残っている
  - `問題箇所`: 人物チェック機能が今回の設定で機能していない
  - `改善提案`: {正しい名前}という名前に直す

**実際のプロンプト**
```
正しい名前で表示されることを目指していたのに、{誤った名前}という名前が残ってしまった。これは人物チェック機能が他のポッドキャストでは機能していたのに、今回の設定で機能していないのが原因かもしれない。{正しい名前}という名前に直すことは可能ですか？
```
