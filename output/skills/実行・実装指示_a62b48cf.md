## テンプレートプロンプト（抽象）

```
{依頼内容}について、{具体的なアクション}を{実行条件}で{追加要件}してください。{注意点や補足}
```

## 変数一覧

- `{{依頼内容}}`: 実装・実行したい対象の概要（例: 音声化機能、比較記事、モデル比較スクリプト）
- `{{具体的なアクション}}`: 実行してほしい具体的な作業（例: 実装計画、データ取得、スクリプト作成、性能調査）
- `{{実行条件}}`: 実行時の制約や前提条件（例: 複数パターンの考慮、特定モデルとの比較）
- `{{追加要件}}`: 追加で満たすべき要件（例: 情報量を多くする、種々の性能や速度を調査）
- `{{注意点や補足}}`: 注意点や補足事項（例: ファクトチェックの影響確認、必要なら修正）

## 活用シーン

- 音声化機能の実装計画を依頼する
- 比較記事のデータ取得とセクション追加を依頼する
- モデル比較スクリプトの作成と性能調査を依頼する
