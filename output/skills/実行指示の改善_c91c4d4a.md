# 🔷 スキル定義: 実行指示の改善

**説明**: 過去の実行結果を踏まえ、指示を具体的に改善するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 10 件 | **カテゴリ**: 実行指示の改善

---

## テンプレートプロンプト（抽象）

```
前回の{作業内容}を踏まえて、{改善点}を実施してください。{追加条件}
```

## 変数一覧

- `{{作業内容}}`: 前回行った作業の概要（例: 議論、実行、設定変更）
- `{{改善点}}`: 具体的に改善したい内容（例: 検証とステップの構成を発展的に、出力先を変更）
- `{{追加条件}}`: その他の条件や制約（例: 時間計測も、設定ファイルも作成）

## 活用シーン

- 前回の議論を踏まえて章構成を改善したいとき
- 設定ファイルの出力先や読み込み方式を変更したいとき
- 未確認の原因を分析して報告してほしいとき

# 🔶 具体インスタンス

### インスタンス 1: 発展的構成の改善

> 元プロンプト: *全ての改善に取り組んで、もう一度実行してください　今度は検証とステポッドキャストの内容を全６回構成の発展的な内容にしてください*

**変数の値**
  - `作業内容`: 実行
  - `改善点`: 全ての改善に取り組んで、もう一度実行、検証と{コンテンツ名}の内容を全{回数}回構成の発展的な内容に
  - `追加条件`: 

**実際のプロンプト**
```
前回の実行を踏まえて、全ての改善に取り組んで、もう一度実行してください。今度は検証と{コンテンツ名}の内容を全{回数}回構成の発展的な内容にしてください。
```
