# 実行指示型タスク

**説明**: ユーザーの指示に従い、具体的な作業を実行するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 14 件 | **ユニークユーザー**: 11 人 | **カテゴリ**: 実行指示型タスク

---

## テンプレートプロンプト

```
{対象}を{アクション}して、{確認事項}を確認して
```

## 変数一覧

- `{{対象}}`: 実行する対象（例: ポッドキャスト生成、グラフ、価格調査）
- `{{アクション}}`: 実行する操作（例: 生成、取得、探索、デバッグ）
- `{{確認事項}}`: 確認する内容（例: メタデータ、結果、状態）

## 活用シーン

- 新しいデータを取得して分析結果を確認する
- システムのテストを実行してエラーをデバッグする
- コンテンツを生成してメタデータを確認する

## トリガーフレーズ

- `実行してみて`
- `試してみたい`
- `確認して`
- `生成してみて`
- `デバッグして`

## 具体インスタンス

### インスタンス 1: ポッドキャスト生成

> 元プロンプト: *voiceboxをダウンロードしたので、方法序説の最新のrunでポッドキャスト生成してみて*

**変数の値**
  - `対象`: 指定したツール、指定したプロジェクトの最新のrun
  - `アクション`: ポッドキャスト生成
  - `確認事項`: 生成結果

**実際のプロンプト**
```
指定したツールをダウンロードしたので、指定したプロジェクトの最新のrunでポッドキャスト生成してみて
```

---

### インスタンス 2: 価格調査と最適化

> 元プロンプト: *日付をずらした場合にどうなるのか 18日から29日など 前後３日間の値段も取得して 日程も伸び縮みさせながら 総合的に最もよいものを探したい これまでの条件などはそのままに、日付ベースで探索してみて*

**変数の値**
  - `対象`: 日付指定した日付範囲、前後指定した日数の値段、日程の伸縮
  - `アクション`: 価格取得、探索、最適化
  - `確認事項`: 総合的に最もよいもの

**実際のプロンプト**
```
日付をずらした場合にどうなるのか 指定した日付範囲など 前後指定した日数の値段も取得して 日程も伸び縮みさせながら 総合的に最もよいものを探したい これまでの条件などはそのままに、日付ベースで探索してみて
```
