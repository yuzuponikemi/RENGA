# 🔷 スキル定義: 指示の修正・最適化

**説明**: ユーザーの作業指示を分析し、汎用的なスキルテンプレートに変換する
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **カテゴリ**: 指示の修正・最適化

---

## テンプレートプロンプト（抽象）

```
{作業内容}について、{改善点}を改善したい。{制約条件}を考慮しつつ、{目標品質}を達成するために{具体的な手法}を用いて{期待成果}を実現してください。また、{追加検討事項}についても検討してください。
```

## 変数一覧

- `{{作業内容}}`: 修正・最適化したい対象の作業やプロジェクト（例: Podcastの台本作成）
- `{{改善点}}`: 現在の問題点や不足している要素（例: 内容が薄い、尺が短い）
- `{{制約条件}}`: 使用可能なリソースや禁止事項（例: 特定のモデルは使わない、時間制限）
- `{{目標品質}}`: 達成したい品質や基準（例: 各回10分のリッチな内容）
- `{{具体的な手法}}`: 使用するツールや手法（例: 特定のモデルを使う、コードを書く）
- `{{期待成果}}`: 最終的に得たい結果（例: 完成度の高いPodcastエピソード）
- `{{追加検討事項}}`: 追加で調査・検討すべき項目（例: モデルとコンテキストウィンドウの関係）

## 活用シーン

- コンテンツ制作の内容充実化と尺調整の指示
- 使用するAIモデルの変更と性能比較の依頼
- システム設定の最適化とベンチマーク取得の指示
