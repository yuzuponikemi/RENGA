# 🔷 スキル定義: 作業の具体化・実行依頼

**説明**: ユーザーの曖昧な指示を具体化し、実行を依頼するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 14 件 | **カテゴリ**: 指示の具体化・実行依頼

---

## テンプレートプロンプト（抽象）

```
以下の内容について、{作業対象} に関して {具体的な作業内容} を実施してください。{追加の条件や制約} を考慮し、{期待する成果物の形式} で出力してください。{補足情報や背景} も踏まえて対応をお願いします。
```

## 変数一覧

- `{{作業対象}}`: 作業の対象となるもの（例: 特定の機能、公開ブログポスト、システムのテスト）
- `{{具体的な作業内容}}`: 実行してほしい具体的な作業（例: 登録方法の調査、関連ポストのピックアップ、テスト実行と確認）
- `{{追加の条件や制約}}`: 作業における条件や制約（例: 無料範囲で、一般公開可、後方互換性を意識）
- `{{期待する成果物の形式}}`: 出力してほしい形式（例: まとめ記事、表付きレポート、手順書）
- `{{補足情報や背景}}`: 作業の背景や補足情報（例: 初めてのツール、小さいデータセットで実験済み）

## 活用シーン

- 新しいツールのセットアップ方法を調査し、手順書としてまとめてほしい
- 複数のデータソースから特定テーマの記事を抽出し、一覧表にしてほしい
- 実験結果を技術記事として複数回に分けて執筆してほしい
