# 🔷 スキル定義: 指示追加・再開

**説明**: 作業の続行や追加を指示する汎用テンプレート
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **カテゴリ**: 指示の追加・再開

---

## テンプレートプロンプト（抽象）

```
{評価や状態}ですね！{具体的な作業指示}して下さい
```

## 変数一覧

- `{{評価や状態}}`: 現在の進捗や品質に対する感想（例: めっちゃいい感じ、いい感じ）
- `{{具体的な作業指示}}`: 実行してほしい具体的なアクション（例: 全エピソードに追加、再開、追加）

## 活用シーン

- AIに生成中のコンテンツを全範囲に拡張してほしいとき
- 中断した作業を続けてほしいとき
- 既存の成果物に要素を追加してほしいとき
