## テンプレートプロンプト（抽象）

```
{指示内容} で {対象} を {アクション} して、出力を確認してください
```

## 変数一覧

- `{{指示内容}}`: 実行すべき具体的な指示や条件（例: 特定のパラメータを設定する、特定の処理モードで実行する）
- `{{対象}}`: 処理の対象となるもの（例: 参加者データ、動画、スクリプト）
- `{{アクション}}`: 実行する操作（例: 生成、検知、試す）

## 活用シーン

- 新しいパラメータでAIモデルを再実行し、結果を確認する
- 特定の動画が正しく検知されるかテストする
- 短いスクリプトで処理の動作確認を行う
