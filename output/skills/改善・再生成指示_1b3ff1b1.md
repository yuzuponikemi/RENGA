# 🔷 スキル定義: 改善・再生成指示テンプレート

**説明**: 既存成果へのフィードバックと再生成・改善を指示するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 7 件 | **カテゴリ**: 改善・再生成指示

---

## テンプレートプロンプト（抽象）

```
{評価コメント}。{具体的な改善指示}。{追加アクション}。{完了条件}。
```

## 変数一覧

- `{{評価コメント}}`: 現状の成果物に対する肯定的・否定的なフィードバック（例: 「いい感じですね」「内容よかったです」）
- `{{具体的な改善指示}}`: 何をどう改善・再生成するかの具体的な指示（例: 「指定したテーマについて調査して記事にまとめて」）
- `{{追加アクション}}`: 改善後に実行すべき追加タスク（例: 「コミット」「プッシュ」「動作確認」）
- `{{完了条件}}`: タスク完了の合図や確認方法（例: 「完了したらプッシュしてください」「動作確認するのがいいと思います」）

## 活用シーン

- 生成した記事やドキュメントの内容をブラッシュアップしたいとき
- ポッドキャスト原稿や解説記事の続編を生成してほしいとき
- アプリの動作確認やバグ修正を指示するとき
