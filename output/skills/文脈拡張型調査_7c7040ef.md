# 🔷 スキル定義: 文脈拡張型調査テンプレート

**説明**: 既存の文脈や外部情報を基に、調査範囲を広げて改善案を生成するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **カテゴリ**: 文脈拡張型調査

---

## テンプレートプロンプト（抽象）

```
{調査対象}は多方面で{情報源}を繰り返して、{背景情報}など取得しています。その前提で、この{ワークフロー名}も同じように{基準文書}以外の視点に話を広げて、{改善候補リスト}どの改善も取り入れたいです。{関連文献}以外の本についても成り立つように考えながら改善プランを練ってください。
```

## 変数一覧

- `{{調査対象}}`: 調査の主体やシステム名
- `{{情報源}}`: 情報収集に使う手段
- `{{背景情報}}`: 取得したい背景やコンテキスト
- `{{ワークフロー名}}`: 改善したいワークフローの名称
- `{{基準文書}}`: 現在の基準となっている文書や視点
- `{{改善候補リスト}}`: 検討したい改善案のリスト
- `{{関連文献}}`: 参照する文献や書籍

## 活用シーン

- 既存の調査レポートを基に、別の視点から改善案をブレインストーミングする
- 複数のユーザーの会話ログを分析し、連続性を追跡する方法を探る
- 特定の書籍や理論を超えて、汎用的な改善プランを立案する
