# 🔷 スキル定義: 更新通知ログの要約

**説明**: ツールやシステムの更新通知ログを要約・解説するスキル
**ステータス**: 🔒 審査中 | **ソース数**: 4 件 | **カテゴリ**: 更新通知の要約

---

## テンプレートプロンプト（抽象）

```
以下の{ログ種類}ログについて、{依頼内容}:

{ログ内容}
```

## 変数一覧

- `{{ログ種類}}`: ログの種類や出典（例: 更新通知、実行ログ）
- `{{依頼内容}}`: ユーザーがログに対してしてほしいこと（例: 解説して、意味を教えて）
- `{{ログ内容}}`: 実際のログテキスト全文

## 活用シーン

- パッケージ管理ツールのバージョン更新通知を要約してほしいとき
- 自動化ツールの実行ログの意味を解説してほしいとき
- システムのメッセージログをわかりやすく説明してほしいとき
