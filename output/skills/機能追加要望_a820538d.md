# 🔷 スキル定義: LLMバックエンド切替機能

**説明**: TelegramコマンドでLLMバックエンドを切り替える機能追加要望
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **カテゴリ**: 機能追加要望

---

## テンプレートプロンプト（抽象）

```
今は{現在のLLMサービス}の{契約プラン}からLLM呼び出しているけど、リミットがかかった時のために{代替LLMモード}も欲しいです　{プラットフォーム}から送るコマンドで切り替えられるようにできるかな？
```

## 変数一覧

- `{{現在のLLMサービス}}`: 現在利用中のLLMサービス名（例: claude, openai, gemini）
- `{{契約プラン}}`: 現在の契約プラン名（例: proプラン, plusプラン, free）
- `{{代替LLMモード}}`: リミット時に切り替えたい代替モード名（例: ollamaモード, localモード, fallbackモード）
- `{{プラットフォーム}}`: コマンドを送信するプラットフォーム名（例: テレグラム, discord, slack）

## 活用シーン

- 現在のLLMサービスのリミット時に代替LLMモードへ自動切替
- APIのレート制限時にローカルモデルへフォールバック
- 無料枠超過時に別サービスへ手動切替
