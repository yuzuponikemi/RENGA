## テンプレートプロンプト

```
{作業内容}について、{具体的な指示}を行ってください。{追加条件}があれば考慮してください。{確認事項}を確認し、{結果報告}をお願いします。
```

## 変数一覧

- `{{作業内容}}`: 実行する作業の種類（例: 環境セットアップ、ファイル作成、調査）
- `{{具体的な指示}}`: 作業の詳細な手順や方法（例: 指定したツールで環境セットアップ、設定ファイルを作成）
- `{{追加条件}}`: 作業に付随する条件や注意点（例: ドキュメントも更新、関連ファイルも整備）
- `{{確認事項}}`: 確認すべき項目（例: 進捗、状況、原因）
- `{{結果報告}}`: 作業後の報告方法（例: 教えてください、保存しておいてください）

## 活用シーン

- 新しいプロジェクトの環境セットアップを依頼する
- 設定ファイルの作成や修正を指示する
- 作業の進捗確認や原因調査を依頼する
