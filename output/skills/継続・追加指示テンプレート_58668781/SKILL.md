# 継続・追加指示テンプレート

**説明**: 過去の作業や生成物に対して継続や追加を指示するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **ユニークユーザー**: 3 人 | **カテゴリ**: 継続・追加指示

---

## テンプレートプロンプト

```
{作業対象}に対して{指示内容}を{追加対象}に{実行範囲}してください
```

## 変数一覧

- `{{作業対象}}`: 継続・追加したい対象（例: 生成された小説、コード、デザイン）
- `{{指示内容}}`: 具体的な作業内容（例: 追加、再開、修正）
- `{{追加対象}}`: 追加する要素（例: 全エピソード、該当箇所、次の章）
- `{{実行範囲}}`: 作業を適用する範囲（例: 全て、一部、特定箇所）

## 活用シーン

- AIに生成してもらった小説の全エピソードに特定の要素を追加したい
- 中断していたコードレビューの再開を依頼したい
- 過去の会話で生成した資料に新しいデータを追加したい

## トリガーフレーズ

- `追加して`
- `続けて`
- `再開して`
- `全部に追加して`
- `続きをお願い`
