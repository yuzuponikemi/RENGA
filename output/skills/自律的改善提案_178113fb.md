# 🔷 スキル定義: 自律的改善提案

**説明**: 過去ログや現状を分析し、改善案や拡張アイデアを提案するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 7 件 | **カテゴリ**: 自律的改善提案

---

## テンプレートプロンプト（抽象）

```
{対象物}を俯瞰して、{分析観点}に基づき、{出力形式}で{アクション}してください。必要に応じて{追加指示}も考慮してください。
```

## 変数一覧

- `{{対象物}}`: 分析・改善の対象（例: 既存のアプリケーション、過去ログ、ワークフロー）
- `{{分析観点}}`: どのような視点で分析するか（例: 拡張可能性、問題点、進捗状況）
- `{{出力形式}}`: 結果の出力方法（例: 記事、一覧、idea.mdファイル）
- `{{アクション}}`: 具体的な行動（例: まとめる、プッシュする、調べる）
- `{{追加指示}}`: その他の補足要件（例: 日本語で、見やすく工夫して）

## 活用シーン

- 既存アプリの機能拡張アイデアを提案する
- 過去の作業ログから改善点を抽出しレポート化する
- プロジェクトの進捗を確認し次のステップを提案する
