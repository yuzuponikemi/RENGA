# 🔷 スキル定義: 設定・変更の指示テンプレート

**説明**: 作業環境や機能の設定変更を指示する汎用テンプレート
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **カテゴリ**: 設定・変更の指示

---

## テンプレートプロンプト（抽象）

```
{操作対象}は{変更内容}に設定してください。{補足条件}がある場合は{条件詳細}を考慮してください。また、{追加依頼}があれば{追加詳細}もお願いします。
```

## 変数一覧

- `{{操作対象}}`: 設定変更の対象（例: 特定の使用モード、検索機能、ワークスペース名）
- `{{変更内容}}`: 具体的な変更内容（例: 常に特定のモードを使う、デフォルトを通常の検索に戻す）
- `{{補足条件}}`: 変更に伴う条件や例外（例: 特別に指定した場合のみ、当面は使わない）
- `{{条件詳細}}`: 補足条件の具体的な説明（例: 特別に指定した場合にのみ特定の機能を使う）
- `{{追加依頼}}`: 変更以外に依頼する追加作業（例: ワークスペース作成、概要追記、変更内容のまとめ）
- `{{追加詳細}}`: 追加依頼の具体的な内容（例: ワークスペース名は指定した名前、変更内容の簡単なまとめ）

## 活用シーン

- AIアシスタントの動作モードを特定のベースに切り替えたいとき
- 検索機能のデフォルト設定を変更したいとき
- 新しい機能追加に伴う設定変更の説明を依頼したいとき

# 🔶 具体インスタンス

### インスタンス 1: CLI常時使用設定

> 元プロンプト: *OK 私は常に特定のモードを使います　このワークスペースは指定した名前と名付けて作成しておいて　概要の追記もお願い*

**変数の値**
  - `操作対象`: 特定の使用モード
  - `変更内容`: 常にそのモードを使う
  - `補足条件`: なし
  - `条件詳細`: なし
  - `追加依頼`: ワークスペース作成と概要追記
  - `追加詳細`: ワークスペース名は指定した名前で作成し、概要も追記

**実際のプロンプト**
```
{操作対象}は{変更内容}に設定してください。補足条件は特にありません。また、ワークスペース作成の追加依頼があればワークスペース名は指定した名前で作成し、概要の追記もお願いします。
```

  - `条件詳細`: 特別に指定した場合にのみその機能を使う
  - `追加依頼`: なし


> 元プロンプト: *今回の変更で、特定の機能を使った検索が可能になったけど　変更内容を簡単にまとめて教えてくれますか　自分の狙いとしては、エージェントの軌道のタイミングで　通常検索するか　特定の機能を使うかどうかを切り替えれるようにしたかった*

**変数の値**
  - `操作対象`: 特定の機能を使った検索機能
  - `変更内容`: 変更内容を簡単にまとめて教える
  - `補足条件`: エージェントの起動タイミングで切り替え可能

  - `追加依頼`: 変更内容の簡単なまとめ
  - `追加詳細`: 変更内容の簡単なまとめ

**実際のプロンプト**
```
{操作対象}は今回の変更で可能になりました。変更内容を簡単にまとめて教えてください。補足条件としてエージェントの起動タイミングで通常検索か特定の機能かを切り替えられるようにしたいです。追加依頼として変更内容の簡単なまとめをお願いします。
```
