# 🔷 スキル定義: 設計レビュー依頼

**説明**: 設計や成果物のレビュー・修正を依頼するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 5 件 | **カテゴリ**: 設計レビュー依頼

---

## テンプレートプロンプト（抽象）

```
{プロジェクト名} の {ブランチ名} ブランチについて、{レビュー対象} をレビュー・修正してください。{追加指示}
```

## 変数一覧

- `{{プロジェクト名}}`: 対象のプロジェクト名（例: あなたのプロジェクト名）
- `{{ブランチ名}}`: レビュー対象のブランチ名（例: 指定したブランチ名）
- `{{レビュー対象}}`: レビューしてほしい具体的な対象（例: 全体の設計、データ形式、ファイル）
- `{{追加指示}}`: その他の補足指示（例: git管理しないように整理、コミットしてください）

## 活用シーン

- プロジェクトの設計全体をレビューしてほしいとき
- 特定のブランチのコードやデータをレビューしてほしいとき
- ファイルの整理やコミットなど、作業の指示を含めて依頼したいとき
