# 🔷 スキル定義: 質問・提案の抽出

**説明**: ユーザーの発言から質問や提案を抽出し整理するスキル
**ステータス**: ✅ 公開中 | **ソース数**: 6 件 | **カテゴリ**: 質問・提案の抽出

---

## テンプレートプロンプト（抽象）

```
以下のユーザーの発言から、質問と提案をそれぞれ抽出してください。

発言: {user_input}

出力形式:
- 質問: {question_format}
- 提案: {proposal_format}

注意: {additional_instructions}
```

## 変数一覧

- `{{user_input}}`: 分析対象のユーザー発言（例: 「このモデルはどれを使うべき？」）
- `{{question_format}}`: 質問の出力形式（例: 「質問内容を箇条書きで列挙」）
- `{{proposal_format}}`: 提案の出力形式（例: 「提案内容を箇条書きで列挙」）
- `{{additional_instructions}}`: 追加の指示や制約（例: 「抽象度を3段階で分類して」）

## 活用シーン

- 会議の議事録からアクションアイテムを抽出する
- チャットログから技術的な質問を抽出する
- フィードバックから改善提案を抽出する

# 🔶 具体インスタンス

### インスタンス 1: モデル選択の質問抽出

> 元プロンプト: *結果として我々はどのモデルを使うべきでしょうか？*

**変数の値**
  - `user_input`: 結果として我々はどのモデルを使うべきでしょうか？
  - `question_format`: 質問内容を箇条書きで列挙
  - `proposal_format`: 提案内容を箇条書きで列挙
  - `additional_instructions`: 質問は1つに絞り、提案がない場合は「なし」と記載

```
以下のユーザーの発言から、質問と提案をそれぞれ抽出してください。
```
