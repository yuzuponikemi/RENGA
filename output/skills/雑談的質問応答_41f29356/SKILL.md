## テンプレートプロンプト

```
たとえばこれですが、わたしの複数の{デバイス名}の{ツール名}ログのスキルは別々の{主体}からのものとして登録できますかね？　それも１つの知見なのですが
```

## 変数一覧

- `{{デバイス名}}`: 使用しているデバイスの種類（例: PC, MacBook, スマホ）
- `{{ツール名}}`: 使用しているAIツールの名前（例: Claude Code, ChatGPT, Copilot）
- `{{主体}}`: データの帰属主体（例: 人間, ユーザー, アカウント）

## 活用シーン

- 複数デバイスでのAIツールログ管理方法を質問する
- AIツールの実行中にデバイスを閉じた場合の挙動を確認する
- 契約プランに基づく認証の可否を質問する
