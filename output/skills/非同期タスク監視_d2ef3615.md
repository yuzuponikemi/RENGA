# 🔷 スキル定義: 非同期タスク監視スキル

**説明**: バックグラウンドタスクの進捗を監視し、完了時に通知する
**ステータス**: 🔒 審査中 | **ソース数**: 22 件 | **カテゴリ**: 非同期タスク監視

---

## テンプレートプロンプト（抽象）

```
Check the status of background task {task_id} ({task_description}). Use TaskOutput with block=false to see current progress. If {stage1_name} is still running and under {stage1_threshold} min, schedule another wakeup in {wakeup_interval_short}s. If {stage1_name} is over {stage1_threshold} min or {stage2_name} started, schedule {wakeup_interval_long}s wakeup. If the full pipeline finished or failed, read {output_path} for full details, summarize, and push a notification to the user.
```

## 変数一覧

- `{{task_id}}`: 監視対象のバックグラウンドタスクID
- `{{task_description}}`: タスクの簡単な説明
- `{{stage1_name}}`: 最初の処理ステージ名
- `{{stage1_threshold}}`: 最初のステージの閾値（分）
- `{{wakeup_interval_short}}`: 短いウェイクアップ間隔（秒）
- `{{stage2_name}}`: 2番目の処理ステージ名
- `{{wakeup_interval_long}}`: 長いウェイクアップ間隔（秒）
- `{{output_path}}`: タスク出力ファイルのフルパス

## 活用シーン

- メディアファイルの文字起こし・話者分離パイプラインの進捗監視
- 長時間の動画エンコード処理の完了待ち
- バッチデータ処理ジョブの状態確認と結果通知
