# 🔷 スキル定義: 非同期タスク進捗監視

**説明**: バックグラウンドタスクの進捗を確認し、結果を通知する
**ステータス**: ✅ 公開中 | **ソース数**: 3 件 | **カテゴリ**: 非同期タスク進捗監視

---

## テンプレートプロンプト（抽象）

```
Check the status of background task {task_id} ({task_name}). Use TaskOutput with block=false. Report which iteration and score if visible. If still running, schedule {wakeup_seconds}s wakeup. If pipeline finished or failed, read {output_file_path} for full details, summarize final score and key topics, and push notification to user.
```

## 変数一覧

- `{{task_id}}`: 監視対象のバックグラウンドタスクID
- `{{task_name}}`: タスクの表示名または説明
- `{{wakeup_seconds}}`: 再チェックまでの待機時間（秒）
- `{{output_file_path}}`: タスク完了時に読み込む出力ファイルのフルパス

## 活用シーン

- バックグラウンドジョブの進捗確認と結果取得
- データ処理パイプラインの実行状況監視と通知
- バッチ処理の完了チェックと成果物確認
