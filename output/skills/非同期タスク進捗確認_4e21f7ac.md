# 🔷 スキル定義: 非同期タスク進捗確認

**説明**: バックグラウンドタスクの状態確認と結果取得を行う
**ステータス**: ✅ 公開中 | **ソース数**: 4 件 | **カテゴリ**: 非同期タスク進捗確認

---

## テンプレートプロンプト（抽象）

```
Check the status of background task {task_id} ({task_name}). Use TaskOutput with block=false. Report iteration number and score. If still running, schedule {wakeup_seconds}s wakeup. If pipeline finished or failed, read {output_path} for full details, summarize final score and key topics, and push notification to user.
```

## 変数一覧

- `{{task_id}}`: バックグラウンドタスクのID（例: ランダムな英数字）
- `{{task_name}}`: タスクの表示名（例: プロジェクト名と処理種別）
- `{{wakeup_seconds}}`: 再確認までの待機秒数（例: 120）
- `{{output_path}}`: 結果ファイルのフルパス（例: /tmp/.../tasks/{task_id}.output）

## 活用シーン

- ファクトチェックパイプラインの進捗確認
- データ分析ジョブの完了待ち
- バッチ処理の状態監視と結果通知
